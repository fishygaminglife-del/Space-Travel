# Space Travel
Space Travel is a fun interactive 2D platformer game where the player has to explor through different planets, gain gear, and defeat aliens.

### Features
- Different Types of Aliens
- Multiple Appealing Maps
- Sound Effects
- Shop Items

### Controls
- WASD or Arrows keys to move
- Control to use Shield
- Escape to Pause
- Mouse to click items
  
 ### Levels
- Space Travel currently consists of 3 levels
- Earth, (Starting Planet)
- Mars, (Fiery Planet)
- Neptune, (Cold Planet)
